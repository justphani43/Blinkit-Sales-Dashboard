# Blinkit-Sales-Dashboard
Built an interactive sales dashboard for Blinkit to analyze business performance and trends. Visualized key KPIs such as total sales ($1.20M), average sales, number of items, and customer ratings. Analyzed product categories, fat content, and outlet performance across different sizes and locations.
:

📊 Blinkit Sales Dashboard (Power BI)
📌 Project Overview

This project presents an interactive sales dashboard built using Power BI to analyze business performance for Blinkit (India’s Last Minute Delivery App). The dashboard provides insights into sales trends, product performance, and outlet-level analysis to support data-driven decision-making.

🎯 Objectives
Analyze overall sales performance and key KPIs
Identify top-performing product categories
Evaluate outlet performance by size, type, and location
Understand customer preferences (fat content, ratings)
📈 Key Insights
Total Sales: $1.20M with an average sales value of $141
Highest sales contribution from fruits, snacks, and household items
Tier 3 outlets generated the highest revenue among all locations
Regular fat products contributed more sales than low-fat products
Supermarket Type1 outlets had the highest overall sales
🛠️ Tools & Technologies
Power BI (Dashboard Development & Visualization)
Power Query (Data Cleaning & Transformation)
DAX (Measures & Calculations)
Excel / CSV (Data Source)
📊 Dashboard Features
KPI Cards: Total Sales, Average Sales, Number of Items, Ratings
Sales Trend Analysis (Outlet Establishment Over Time)
Product Category Performance
Outlet Analysis (Size, Location Tier, Type)
Fat Content Analysis
Interactive Filters (Slicers) for dynamic exploration
🧩 Data Model
Implemented a star schema for efficient data modeling
Fact table: Sales data
Dimension tables: Outlet, Item Type, Location, Fat Content
🚀 How to Use
Download the .pbix file from this repository
Open using Power BI Desktop
Use slicers to filter data by outlet size, type, and item category
Explore visuals for insights
